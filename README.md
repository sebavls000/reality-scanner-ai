# 🧠 Reality Scanner AI

Detectá manipulación, narrativa y sesgo en segundos.

Reality Scanner AI es una aplicación interactiva desarrollada con Python y Streamlit que permite analizar textos, detectar patrones de manipulación y comparar discursos de forma visual, rápida y clara.

---

## 🚀 Demo en vivo

👉 https://reality-scanner-ai-sebavls000.streamlit.app

Accedé directamente a la app desde el navegador sin instalar nada.

---

## ✨ Qué hace esta app

Reality Scanner AI permite:

- analizar textos individuales  
- detectar manipulación narrativa  
- clasificar el tipo de discurso (ciencia, religión, filosofía, manipulación)  
- comparar dos textos entre sí  
- visualizar resultados con gráficos  
- ver tendencias de análisis  
- explorar un dataset de ejemplo  

---

## 🧩 Funcionalidades principales

### 🔍 Análisis individual

Ingresás un texto y la app devuelve:

- clasificación automática  
- nivel de manipulación  
- métricas por categoría  
- visualización gráfica  

---

### ⚔️ Comparador de narrativas

Compará dos textos y descubrí:

- cuál es más manipulador  
- diferencias entre ambos  
- puntuación comparativa  
- gráfico visual  

---

### 🔥 Tendencias

La app guarda los análisis realizados y muestra:

- historial de textos  
- score de manipulación  
- evolución visual  

---

### 📊 Dataset de ejemplo

Incluye textos precargados para probar la app sin necesidad de cargar datos externos.

## 🧪 Ejemplos de uso

Ejemplo 1 – Análisis individual

Texto:
La élite controla la verdad y manipula a las masas

Resultado esperado:
- Manipulación alta  
- Clasificación: Manipulación  

---

Ejemplo 2 – Análisis individual

Texto:
La ciencia se basa en evidencia y análisis de datos

Resultado esperado:
- Baja manipulación  
- Clasificación: Ciencia  

---

Ejemplo 3 – Comparación

Texto A:
La ciencia se basa en evidencia y análisis de datos

Texto B:
La élite controla la verdad y manipula a las masas

Resultado esperado:
- Texto B más manipulador  
- Diferencia clara en el gráfico

## ▶️ Cómo correrlo localmente

1. Clonar el repositorio:

git clone https://github.com/sebavls000/reality-scanner-ai.git  
cd reality-scanner-ai  

2. Instalar dependencias:

pip install -r requirements.txt  

3. Ejecutar la app:

streamlit run app.py
