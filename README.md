# 🧠 Reality Scanner AI

> Detectá manipulación, narrativa y sesgo en segundos.

## 🚀 Qué es esto

Reality Scanner AI es una app interactiva para analizar textos y detectar:

- manipulación narrativa
- sesgos ideológicos
- patrones de discurso
- diferencias entre textos

## ⚡ Funciones

- análisis individual
- comparador de narrativas
- historial de resultados
- dataset de ejemplo
- visualización con gráficos

## 🛠️ Tecnologías

- Python
- Streamlit
- Pandas
- Plotly

## ▶️ Cómo correrlo

```bash
pip install -r requirements.txt
streamlit run app.py
